# Cinnabar REST API Source Code

I build free REST API based on Golang. I try to apply MVC  (CodeIgniter 4 in PHP) concept on Golang

## Response Format

### 200 OK
```json
{
    "data": any;
}
```
### 400 Bad Request
```json
{
    "errors": any;
}
```
### 500 Internal Server Error
```json
{
    "errors": any;
}
```
